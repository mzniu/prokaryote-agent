# ⚠️ 核心酶层 - 禁止自主修改 ⚠️

## 警告

此目录包含原核生物Agent的**核心酶（Core Enzymes）**代码。

核心酶是Agent的"DNA转录机制"，负责生成所有其他技能代码。
如同生物体的DNA聚合酶，核心酶是生命运作的基础，**绝对不能被自主修改**。

## 规则

1. **只读访问** - Agent只能调用核心酶，不能修改
2. **上帝模式** - 只有开发者（人工）可以修改此目录
3. **版本锁定** - 每次人工修改需更新版本号
4. **完整性校验** - 核心酶运行时会自检，防止篡改

## 文件列表

| 文件 | 用途 | 可修改 |
|------|------|--------|
| base.py | CoreEnzyme基类 | ❌ |
| code_synthase.py | 代码合成酶 | ❌ |
| syntax_verifier.py | 语法校验酶 | ❌ |
| error_repairase.py | 错误修复酶 | ❌ |
| pipeline.py | 技能生成管线 | ❌ |

## 如果需要修复

如果核心酶出现问题，请：
1. 联系开发者进行人工诊断
2. 在上帝模式下修改代码
3. 更新 CORE_ENZYME_VERSION
4. 运行完整性测试
