# 进化目标 (Evolution Goals)

> 在此文件中定义你希望智能体进化出的能力目标。智能体会按顺序执行每个目标，直到全部完成。
> 使用 Markdown 格式，每个一级标题 `#` 代表一个进化目标。

---

## 格式说明

每个目标使用以下格式：

```markdown
# 目标标题
- **状态**: pending | in_progress | completed | failed
- **优先级**: high | medium | low
- **描述**: 详细描述这个能力应该做什么

### 验收标准
- [x] 标准1
- [x] 标准2
```


---

# 网络搜索能力
- **状态**: completed
- **优先级**: high
- **描述**: 创建网络搜索功能，能够从互联网获取最新资讯、新闻、技术文档等信息。支持搜索引擎查询，并提取关键信息返回给用户。

### 验收标准
- [x] 能够通过搜索引擎查询关键词
- [x] 返回相关的搜索结果（标题、链接、摘要）
- [x] 支持获取网页内容并提取关键信息
- [x] 能够处理不同类型的查询（新闻、技术、常识等）
- [x] 正确处理网络错误和超时情况
- [x] 返回格式清晰，便于用户理解

---

# 软件开发架构能力
- **状态**: completed
- **优先级**: high
- **描述**: 创建软件架构设计和分析能力，能够根据需求生成系统架构方案、技术选型建议、模块划分方案等。支持分析现有代码结构，提供重构建议和最佳实践指导。

### 验收标准
- [x] 能够根据需求描述生成系统架构方案（如分层架构、微服务架构等）
- [x] 提供技术栈选型建议（语言、框架、数据库等）
- [x] 生成模块划分和组件设计方案
- [x] 能够分析现有代码结构并识别架构模式
- [x] 提供代码重构建议和优化方向
- [x] 输出清晰的架构文档（Mermaid图表、文字说明）
- [x] 支持多种架构风格（MVC、MVVM、Clean Architecture、DDD等）
- [x] 能够评估架构方案的优缺点和适用场景
